默认使用官方Pico开发板

-----------

`pico-datasheet <https://datasheets.raspberrypi.com/pico/pico-datasheet.pdf>`_

`pico-started <https://datasheets.raspberrypi.com/pico/getting-started-with-pico.pdf>`_


编译环境
-----------

.. code-block:: bash

    sudo apt-get update
    sudo apt-get install -y gcc-arm-none-eabi cmake libnewlib-arm-none-eabi libstdc++-arm-none-eabi-newlib git


rust
~~~~~~~~~~~

`rp2040-rust-template <https://github.com/SoCXin/RP2040/tree/master/templates/rust>`_

.. code-block:: bash

    rustup target install thumbv6m-none-eabi
    cargo install flip-link
    # This is our suggested default 'runner'
    cargo install probe-run
    # If you want to use elf2uf2-rs instead of probe-run, instead do...
    cargo install elf2uf2-rs --locked
